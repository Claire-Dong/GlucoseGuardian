To build and run Insertion Sort:
make asn1_a
./asn1_a size             where size is size of array

To build and run Merge Sort:
make asn1_b
./asn1_b size              where size is size of array

To build and run part c) Sort:
make asn1_c
./asn1_c size k            where size is size of array 
                           and insertion sort is run if size <= k else merge sort

To remove all object files and executables:
make clean
